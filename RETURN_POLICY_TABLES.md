## 9. Special Cases & Handling

| Case Type                | Condition                   | Approval Rate | Timeline | Resolution                 |
| ------------------------ | --------------------------- | ------------- | -------- | -------------------------- |
| **Courier Damage**       | Damage during delivery      | High          | 1-2 days | Replacement pre-shipped    |

| **Wrong Item**           | Incorrect product shipped   | Very High     | Same-day | Replacement + return label |

| **Seller Refusal**       | Seller rejected valid claim | Medium        | 2-3 days | Platform mediator decides  |
| **Damage After Review**  | Item damaged post-delivery  | Medium        | 3-5 days | Requires proof (photos)    |

## 12. COD vs Prepaid Return Differences

| Aspect               | COD (Cash on Delivery)         | Prepaid (Card/UPI/Wallet)       |
| -------------------- | ------------------------------ | ------------------------------- |
| **Refund Method**    | Wallet credit or bank transfer | Back to original payment method |
| **Refund Speed**     | 5-7 business days              | 3-5 business days               |
| **Pickup Speed**     | Same day or next day           | 2-3 business days               |
| **Additional Fee**   | None                           | None                            |
| **Proof of Payment** | SMS receipt                    | Order confirmation              |

## 13. Return Eligibility Checklist

**Before you can return an item, verify:**

| Requirement                                                           | Status                   |
| --------------------------------------------------------------------- | ------------------------ |
| ✅ Order status is "DELIVERED"                                        | Required                 |
| ✅ Item is within return window (7 days full / 15 days replacement)   | Required                 |
| ✅ Item is in returnable condition (unused/undamaged for full return) | Required for Full Return |
| ✅ Product is not electronics (or requesting replacement only)        | Required for Full Return |
| ✅ All accessories & documentation included                           | Recommended              |
| ✅ Original packaging intact                                          | Recommended              |
| ✅ Reason is valid (defect/wrong item/etc.)                           | Required                 |
| ✅ Damage photos uploaded if condition is "DAMAGED"                   | Required if Damaged      |

## 13. Refund Status Lookup

| Status        | What It Means                      | Your Action                 | Timeline              |
| ------------- | ---------------------------------- | --------------------------- | --------------------- |
| **INITIATED** | Refund sent to bank/wallet         | Wait for credit             | 3-5 days              |
| **PROCESSED** | Bank is processing refund          | Check bank app              | 3-5 days              |
| **COMPLETED** | Refund received in your account    | Transaction complete        | No action             |
| **DELAYED**   | Refund taking longer than expected | Contact support if >7 days  | Up to 10 days         |
| **FAILED**    | Refund couldn't process            | Contact support immediately | Support to reinitiate |

## 15. Common Return Questions (Quick Answers)

| Question                                     | Answer                                                   |
| -------------------------------------------- | -------------------------------------------------------- |
| **Can I return without original packaging?** | Yes, but approval less likely. Item must be pristine.    |
| **What if I lost the item?**                 | Contact seller immediately. Cannot return what's lost.   |
| **Can I return after 7 days?**               | No for full return. Replacement available up to 15 days. |
| **Does seller always approve?**              | No. Seller reviews photos/condition and decides.         |
| **What if seller rejects unfairly?**         | Appeal through app. Platform mediator will review.       |
| **How fast is wallet refund?**               | Instant (minutes). Card/bank refunds take 3-5 days.      |
| **Do I pay for return shipping?**            | No. Seller arranges and pays for pickup.                 |
| **Can I track my refund?**                   | Yes. Check status in app under Return Status.            |
| **What if refund doesn't arrive?**           | Wait 7 days. Contact bank. Then contact support.         |
| **Can electronics be returned for refund?**  | No. Only replacement available. No exceptions.           |
| **Can I return extracted/opened products?**  | No. Only replacement available for extracted items.      |
